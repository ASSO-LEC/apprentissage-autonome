Les Interrogatifs feuille 2

Entraînez-vous à utiliser : Que, Quel, Quels, Quelle, Quelles

Consignes pour les apprenants :

Choisissez l'interrogatif correct pour chaque phrase.

  • Que (ou Qu') : s'utilise pour poser une question sur une chose (COD du verbe).
  • Quel : s'accorde avec un nom masculin singulier.
  • Quels : s'accorde avec un nom masculin pluriel.
  • Quelle : s'accorde avec un nom féminin singulier.
  • Quelles : s'accorde avec un nom féminin pluriel.
