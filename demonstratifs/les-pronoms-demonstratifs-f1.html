Les Pronoms Démonstratifs feuille 1

Entraînez-vous à utiliser : celui/celle(s)/ceux (-ci/-là), ceci, cela

Consignes pour les apprenants :

Remplacez le groupe nominal répété ou désignez l'objet correct à l'aide d'un pronom démonstratif adapté.

  • Celui-ci / Celui-là : masculin singulier (proche / lointain).
  • Celle-ci / Celle-là : féminin singulier (proche / lointain).
  • Ceux-ci / Ceux-là : masculin pluriel (proche / lointain).
  • Celles-ci / Celles-là : féminin pluriel (proche / lointain).
  • Ceci / Cela (ou Ça) : neutre, remplace une idée, une action ou désigne quelque chose de non identifié.
